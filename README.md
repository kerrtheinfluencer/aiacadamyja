# Futur4IsHere Mobile AI Academy

Single-page mobile-first training app for teaching students how to vibe-code web apps with AI chats (ChatGPT, Gemini, etc.), using one copy-paste prompt and an instant HTML preview lab.

## Run

Open `index.html` directly or serve with:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.
