# Futur4IsHere Mobile AI Academy

Premium single-page mobile-first training app for teaching students how to vibe-code web apps with AI chats (ChatGPT, Gemini, etc.) using one starter prompt, a live preview lab, and an auto-graded class test.

## Run

Open `index.html` directly, or serve with:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.

## Flow

1. Teach Module 1 (prompt basics).
2. Students take the 4-question test.
3. On pass (3/4+), Module 2 and Module 3 auto-unlock and auto-check complete.
4. Students copy the starter prompt and use the live preview lab.
