Mobile-first AI Academy

Teach students to vibe-code web apps from AI chat on phone.

This one-page program is built for mobile learners using ChatGPT, Gemini, and similar AI chats. Students copy one mega-prompt, paste it in chat, and immediately render app previews right in chat.

1) One single copy-and-paste starter prompt

Students paste this once into AI chat, then ask for edits in plain language.

You are my mobile vibe-coding coach.
Build a single-file HTML web app I can run directly in mobile chat previews.
Rules:
1) Return only one complete HTML file between ```html code fences.
2) Include all CSS and JavaScript in the same file.
3) Make the UI mobile-first and touch-friendly.
4) Keep buttons large and labels clear.
5) Add a short "How to edit this app in chat" section at the top.
6) After each version, suggest 3 tiny improvements I can ask for next.
Now build: [describe your app idea here].

2) Practice preview lab (paste generated HTML)

Use this to show students how chat-generated code can instantly render as a live app preview.

Welcome to futur4ishere

This is your quick-start lesson pop-up. Tell students the code word futur4ishere to begin class.

  1. Copy the one-paste prompt.
  2. Open ChatGPT/Gemini on mobile and paste it.
  3. Replace the app idea with your own project goal.
  4. Run each output in the preview lab to test instantly.
  5. Ask AI for one small upgrade at a time.
